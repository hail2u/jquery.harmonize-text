TITLE
=====

jQuery Plugin: Harmonize Text - version 0.1.1


DESCRIPTION
===========

Change font-size of selected elements to harmonize text with their parent element.


Usage
=====

Harmonize all heading element:

    $("h1, h2, h3, h4, h5, h6").harmonizeText();
